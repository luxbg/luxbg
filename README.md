# Hi, I'm Mohamed Hamed

Backend engineer who started working at 14, because apparently normal teenage hobbies were not stressful enough.

Over the years, I have built APIs, payment systems, blockchain infrastructure, wallets, transaction pipelines, and other software responsible for handling people’s money, which is a perfectly reasonable amount of responsibility to give someone who once needed permission to leave school early.

I care about performance, scalability, clean architecture, and making sure financial systems do not accidentally turn into donation platforms.

I have not been working since 2025, so most of my professional work and contributions are still sitting inside private repositories. That means my GitHub activity may look suspiciously quiet, but I promise I did not suddenly forget how computers work.

Most of the interesting code is private, most of the public code is unfinished, and most of the finished personal projects are waiting for me to write a README.

Building things, breaking things, fixing them before anyone notices, and occasionally convincing distributed systems to behave.
